# job4j_kt

Учебный проект на языке Kotlin

## Описание

Репозиторий предназначен для решения задач курса по языку программирования Kotlin
- Преподаватель: Пётр Арсентьев
- Ссылка на курс: https://job4j.ru/courses/22

## Цели курса

Знакомство с языком программирования

## Темы курса

- Первая программа
- Типы данных и методы
- Тестирование
- Операторы условий if
- Циклы while и for
- Массивы и коллекции


## Стек технологий:

 - java 17
 - Kotlin 1.8
